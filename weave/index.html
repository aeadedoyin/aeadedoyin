weave v1.0.0

Every Claude Code session, woven into one window.

Weave is a desktop console for Claude Code. Run several sessions side by side in a chat-style UI, drop into a real terminal when you need it, watch git diffs as they land, and keep parallel work isolated in worktrees. No more juggling terminal windows.

The loom /01
01

Parallel sessions

Run as many Claude Code sessions as you have ideas, side by side in one window. Each is its own long-lived conversation.

02

The real CLI

Weave drives the actual claude binary on your PATH, with your shell environment captured per project. Your hooks, your config, your accounts.

03

Terminal takeover

Any session can drop into a real PTY running claude --resume, then hand back to the chat view. Same session, both worlds.

04

Git, watched

A live diff panel shows what each session is changing as it happens, before anything is committed.

05

Worktree isolation

Point parallel sessions at isolated git worktrees so they never step on each other's changes.

06

Multiple accounts

Personal and work Claude accounts live together via config dirs, with an in-app /login flow per account.

You will need /03
  • A Mac, Apple Silicon or Intel. Windows is not supported: Weave captures your environment through a POSIX login shell.
  • The Claude Code CLI, installed and logged in: npm i -g @anthropic-ai/claude-code. Weave drives the real CLI, so it must be on your shell's PATH.
  • git, optionally, for the diff panel and worktree isolation.

Named after Warren Weaver, Claude Shannon's co-author on The Mathematical Theory of Communication. Also: it weaves threads.
